üöÄ Quick Start Example 

# 1. Train the model (run separately)
python model.py

# 2. Start the API server
python main.py

# 3. Get statistics
curl http://localhost:8002/api/stats

# 4. Make single prediction
curl -X POST -H "Content-Type: application/json" \
  -d '{"hour": 20, "dayofweek": 4, "lag_1h": 2.5, "lag_24h": 2.8, "rolling_mean_24h": 2.3}' \
  http://localhost:8002/api/predict_single
